## Several commonly used

| Name       | File Extension          | Content-Type(Mime-Type)                                      |
| ---------- | ----------------------- | ------------------------------------------------------------ |
| 2010 Excel | .xlsx                   | application/vnd.openxmlformats-officedocument.spreadsheetml.sheet |
| Text       | .txt                    | text/plain                                                   |
| Image      | .png/.jpg/.gif          | image/* eg: (image/png)                                      |
| Page       | .htm/.html              | text/html                                                    |
| Video      | .avi/ .mpg/ .mpeg/ .mp4 | video/*                                                      |
| Audio      | .mp3/ .wav/             | audio/*                                                      |
| PDF        | .pdf                    | application/pdf                                              |
