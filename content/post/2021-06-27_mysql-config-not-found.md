# 安装mysql-devel

centos系统使用yum安装 mysql-devel

```bash
yum install mysql-devel
```

ubuntu 系统apt-get 安装libmysqlclient-dev

```bash
apt-get install libmysqlclient-dev
```

debian 系统

```bash
apt install -y libmariadbd18
apt install -y libmariadbd-dev
```
